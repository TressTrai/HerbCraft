Student gamejam project.

An ancient Slavic cat lives in the forest to save ordinary peasants.
He fulfills various requests from the common folk, such as curing migraines, diarrhea, and other ailments.
However, not everything is so simple. Cursed evil spirit has appeared, preventing the gathering of ingredients!
